##
## Settings

> I was using Google Chrome 103.0.5060.114 

> Also you can find requirements.txt 

1. Install requirements: `pip install -r requirements.txt`

2. Set up the variable: `set FLASK_APP=app.py`

3. Just run: `flask run`
